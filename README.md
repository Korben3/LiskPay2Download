# LiskPay2Download
Simple lisk payment script - pay 2 download an item.

This script allows you to setup a payment gateway for your website.

Libs used: jquery

languages: html, css, javascript, php

usage: edit the configuration file and upload all the files to your server. Be sure to change the name of your Hi Res image directory.

Example: https://liskpay2download.korben3.com

With some minor editing, this method can also be used to sell software and anything else that can be downloaded.
